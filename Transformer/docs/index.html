项目概述 模型架构 训练过程 推理过程 微调指南 数据集准备

项目概述

这是一个基于PyTorch实现的Transformer模型，专门针对英汉机器翻译任务优化。本项目特别适配了Apple Silicon (M4 Pro)芯片，充分利用MPS加速功能。

环境要求

  • MacBook Pro (Apple M4 Pro芯片)
  • 48GB 内存
  • Python 3.8+
  • PyTorch 2.0+

项目结构


Transformer/
├── model.py          # Transformer模型核心实现
├── train.py          # 训练脚本
├── inference.py      # 推理脚本
├── finetune.py       # 微调脚本
├── config.py         # 配置文件
├── data/             # 数据集目录
├── saved_models/     # 模型保存目录
└── docs/             # 文档目录

快速开始

  1. 克隆项目到本地
  2. 安装依赖：pip install -r requirements.txt
  3. 准备数据集：运行python preprocess.py
  4. 开始训练：python train.py
  5. 进行推理：python inference.py
